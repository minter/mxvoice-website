Updating the App

Keep Mx. Voice up to date with the latest features and fixes

{{> docs-sidebar}}

Automatic Updates (macOS and Windows)

On macOS and Windows, Mx. Voice checks for updates automatically when the app launches. If a new version is available, you'll see a notification with the version number and release notes.

Update Flow

  1. A dialog appears showing the new version number and what's changed.
  2. Click Install Now to start the download. You can also click Later to dismiss and update another time.
  3. The button shows download progress (e.g., "Downloading... 45%").
  4. Once the download completes, the app restarts automatically with a short countdown.

Your songs, profiles, hotkeys, and settings are preserved through the update — nothing is lost.

Prerelease Updates

By default, Mx. Voice only offers stable releases. If you'd like to try new features early, you can opt in to prerelease (beta) updates in Preferences by enabling the Prerelease Updates toggle.

If you're already running a prerelease version, the updater will automatically check for newer prereleases regardless of the toggle setting.

Manual Download

You can always download the latest version directly:

  • macOS (DMG) — Universal binary for Intel and Apple Silicon
  • Windows (EXE) — 64-bit installer
  • All releases on GitHub — includes Linux AppImage and .deb packages

Linux Updates

Automatic updates are not available on Linux. To update, download the latest AppImage or .deb package from the GitHub releases page and install it over your existing version.

Release Notes

You can view release notes for any version at any time:

  • In the app, the update dialog shows release notes for the new version.
  • On the web, visit the GitHub releases page.
